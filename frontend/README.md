# تطبيق حجز قاعات الأفراح

تطبيق ويب متكامل لإدارة وحجز قاعات الأفراح والمناسبات، يوفر تجربة سهلة للمستخدمين والإدارة، مع لوحة تحكم متقدمة ودعم للمحادثة الفورية بين العميل والإدارة.

---

## المميزات الرئيسية

### للمستخدمين

- **تصفح القاعات**  
  استعراض جميع القاعات المتاحة مع صور، تفاصيل، تقييمات، وسعر كل قاعة.

- **بحث وتصنيف متقدم**  
  البحث باسم القاعة أو الموقع، مع تصنيف القاعات (فاخرة/اقتصادية/الكل).

- **عرض تفاصيل القاعة**  
  مشاهدة صور متعددة، وصف القاعة، السعة، الموقع على الخريطة، التقييمات، وآراء العملاء.

- **حجز القاعة بسهولة**  
  اختيار التاريخ والوقت وعدد الضيوف، وإضافة ملاحظات أو خدمات إضافية (مثل كوشة، دي جي، بوفيه...).

- **إدارة الحجوزات**  
  عرض جميع الحجوزات السابقة والحالية، مع إمكانية متابعة حالة الحجز (معلقة، مؤكد، عربون، ملغاة).

- **المفضلة**  
  إضافة القاعات المفضلة للوصول السريع لاحقًا.

- **تقييم القاعات وكتابة التعليقات**  
  تقييم القاعة بعد الحجز وكتابة رأي شخصي يظهر لباقي المستخدمين.

- **محادثة مباشرة مع الإدارة**  
  شات فوري مع الإدارة لكل حجز، لمتابعة الطلب أو الاستفسار.

- **إدارة الحساب الشخصي**  
  تعديل الاسم ورقم الجوال، وتسجيل الخروج بسهولة.

---

### للإدارة (لوحة التحكم)

- **إحصائيات شاملة**  
  عدد القاعات، المستخدمين، الحجوزات، وعدد المشاهدات.

- **إدارة الحجوزات**  
  عرض جميع الطلبات مع تفاصيل المستخدم والقاعة، وتغيير حالة الحجز (موافقة، عربون، إلغاء...).

- **إدارة القاعات**  
  إضافة قاعة جديدة، تعديل أو حذف القاعات الحالية.

- **إدارة المستخدمين**  
  عرض جميع المستخدمين المسجلين.

- **إدارة التعليقات**  
  مراجعة التعليقات والتقييمات، مع إمكانية التعديل أو الحذف.

- **شات الإدارة**  
  شاشة خاصة لمتابعة جميع المحادثات مع العملاء، مع تصنيف الحجوزات حسب الحالة.

## طريقة الاستخدام

### 1. تشغيل التطبيق

#### المتطلبات:
- Node.js
- قاعدة بيانات SQLite (تُنشأ تلقائيًا)

#### خطوات التشغيل:

1. **تثبيت الحزم**
   ```bash
   cd backend
   npm install express cors sqlite3
   ```

2. **تشغيل السيرفر**
   ```bash
   node server.js
   ```
   سيعمل السيرفر على:  
   [http://localhost:3001](http://localhost:3001)

3. **فتح الواجهة**
   - للمستخدمين: افتح `frontend/index.html` أو توجه إلى [http://localhost:3001](http://localhost:3001)
   - للإدارة: افتح `frontend/admin.html`
   - شات الإدارة: افتح `frontend/admin-chat.html`

## شرح عمل التطبيق

### 1. تسجيل الدخول/التسجيل
- يسجل المستخدم برقم الجوال وكلمة مرور واسم.
- يتم حفظ بياناته في قاعدة البيانات.

### 2. تصفح القاعات
- تظهر قائمة القاعات مع إمكانية البحث والتصنيف.
- عند الضغط على قاعة تظهر التفاصيل الكاملة.

### 3. الحجز
- يختار المستخدم التاريخ والوقت وعدد الضيوف والخدمات الإضافية.
- يتم إرسال الطلب للإدارة ويظهر في صفحة "حجوزاتي".

### 4. متابعة الحجز
- يمكن للمستخدم متابعة حالة الحجز (معلقة/مؤكدة/عربون/ملغاة).
- يمكنه التواصل مع الإدارة عبر الشات الخاص بالحجز.

### 5. التقييم والتعليق
- بعد انتهاء الحجز يمكن للمستخدم تقييم القاعة وكتابة تعليق يظهر لباقي المستخدمين.

### 6. لوحة تحكم الإدارة
- تعرض جميع الحجوزات مع إمكانية تغيير الحالة.
- إضافة وتعديل القاعات.
- مراجعة التعليقات والتواصل مع العملاء عبر الشات.
